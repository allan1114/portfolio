Modern Portfolio — Brand logo is TEXT, not an image.

The logo renders as "YN." (your initials, two chars + period) with these exact specs:

  text:       "YN."              (customize: replace with user's initials)
  font:       Inter / Segoe UI, weight 800
  font-size:  1.3rem (20.8px) in nav
  letter-spacing: -0.5px

  fill:       background-clip gradient
  background: linear-gradient(135deg, #FF4D6D, #FFBE0B)

  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;

There is no SVG or PNG logo file. The "logo" is a CSS treatment applied
to any 2-3 character string. See preview/card-brand-logo.html for a
rendered example.
